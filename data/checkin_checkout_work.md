# 仕事向けチェックイン・チェックアウトカード

会議やチーム活動の開始・終了時に、**問いを元に全員が一言ずつ発言する**ための問い集です。

---

## 使い方

- **チェックイン**：活動の**最初**に行う。場に入る・集中するための一言。
- **チェックアウト**：活動の**最後**に行う。振り返る・締めるための一言。
- **初級・中級・上級**：場の状況に合わせて選ぶ
  - 初級：初対面・短時間・緊張している場
  - 中級：慣れたメンバー・十分な時間
  - 上級：深く話したい・チームが成熟している

---

## チェックイン（場に入る・集中する）

### 初級
| # | 問い |
|---|------|
| 1 | 今日の調子は？ |
| 2 | この会議への期待は？ |
| 3 | 今一番気になっていることは？ |
| 4 | 今の期待は？ |
| 5 | 今日、一言で自分を表すと？ |
| 6 | この場に来るまでに、どんな一日でしたか？ |
| 7 | 今日、集中したいことは？ |
| 8 | 今日のテンションを1〜10で表すと？ |

### 中級
| # | 問い |
|---|------|
| 1 | 今日、誰の話を特に聴きたいですか？ |
| 2 | 自分が今日貢献できることは？ |
| 3 | 今のチームの雰囲気を一言で表すと？ |
| 4 | 今日、一人ひとりの話を丁寧に聴くために何をしますか？ |
| 5 | この会議で、自分が持ちたい姿勢は？ |
| 6 | 今日の対話で、何を得たいですか？ |
| 7 | このチームに対して、今感じていることは？ |
| 8 | 今日、場に持ち込みたい問いは？ |

### 上級
| # | 問い |
|---|------|
| 1 | このプロジェクトに対して、自分はどう関わりたいですか？ |
| 2 | 今日、対話の中でどんな問いを持ちたいですか？ |
| 3 | このグループは、お互いにどの様に頼り頼られていると感じますか？ |
| 4 | 今日の場を、どういう場にしたいですか？ |

| 6 | このチームにもっと期待していることは？ |
| 7 | 今日の対話を通じて、何を探求したいですか？ |

## チェックアウト（振り返る・締める）

### 初級
| # | 問い |
|---|------|
| 1 | 今日一番言えてよかったことは？ |
| 2 | 明日への引継ぎは？ |
| 3 | 時間通りに進めるために意識したことは？ |
| 4 | 今日の一言を締めくくると？ |
| 5 | 今日の会議、一言で表すと？ |
| 6 | 明日の最初のアクションは？ |
| 7 | 今日、一番印象に残ったことは？ |
| 8 | 終わった今、どんな気持ちですか？ |

### 中級
| # | 問い |
|---|------|
| 1 | 今日、誰の言葉が印象に残りましたか？ |
| 2 | 次に自分が動きたいアクションは？ |
| 3 | チームに伝えたいことは？ |
| 4 | 今、誰にどのようなことを伝えたいですか？ |
| 5 | 今日の話し合いで、自分が変わった点は？ |
| 6 | 今日学んだことを、一言で表すと？ |
| 7 | 次回までに、自分が準備したいことは？ |
| 8 | 今日の場を振り返って、感謝したいことは？ |

### 上級
| # | 問い |
|---|------|
| 1 | 今日の対話は、どう深まったと思いますか？ |
| 2 | 自分の中で変わった視点はありますか？ |
| 3 | このチームにもっと期待したいことは？ |
| 4 | 今日の話し合いはどのように深まったと思いますか？ |
| 5 | 今日の対話で、自分の中に生まれた新しい問いは？ |
| 6 | このチームの対話を、今後どう発展させたいですか？ |
| 7 | 今日の経験を、明日からの仕事にどう活かしますか？ |

---

## 使いドコロ

- 会議の始まりと終わりに
- 1on1の前後
- プロジェクトのキックオフ・振り返り
- チーム朝会・夕会
- リモートワークの始業・終業
- 研修・勉強会の前後

---

## データ形式

問いデータは `checkin_checkout_work.json` に格納されています。アプリやスクリプトから読み込んで利用できます。
